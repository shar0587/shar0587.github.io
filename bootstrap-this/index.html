Brief Overview of Bootstrap

Bootstrap, in a nutshell, is a toolkit combining HTML, CSS, and JavaScript elements / components for the purpose of web development. It was originally created for Twitter, and has ever since grown in popularity upon its open source release back in 2011 due to its flexible and easy-to-work-with nature. With Bootstrap, users are able to create a nice looking website much quicker in comparison to making one from scratch using HTML and CSS.

Breakpoints

There are 6 breakpoints in Bootstrap, which are used as media queries. They go as follows:

  • X-Small (no class name used), less than 576px
  • Small (sm), greater than/equal to 576px
  • Medium (md), greater than/equal to 768px
  • Large (lg), greater than/equal to 992px
  • Extra Large (xl), greater than/equal to 1200px
  • Extra Extra Large (xxl), greater than/equal to 1400px

How to Get Started

1. Bootstrap can be installed in 2 ways, either from bootstrap.com directly as precompiled, or by copying the source code template found on their website into your code text editor. These instructions will follow Visual Studio Code / the source code.
2. When installed, ensure you are using the HTML5 doctype at the beginning of your document (For Visual Studio Code, simply type "!"). Ensure you have the proper meta tags + BootStrap CSS + Optional JavaScript included.
3. Start creating your page using the various predefined tools available on Bootstrap. The great feature about Bootstrap is every element has CSS built in, meaning styling is done by Bootstrap. Visit their website for more detailed instructions on each element capability.
4. Columns and class names are at the forefront with Bootstrap. Class names define how big the element is, what it's styled as, and so on.
